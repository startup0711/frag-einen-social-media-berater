FAQ

Worauf muss ich beim Stellen der Frage achten?

Betreff: Geben Sie einen aussagekräftigen Betreff ein. "Möglichkeiten zur Leadgenerierung für ein mittelständisches Unternehmen im Agrarbereich" grenzt das Thema gut ein. "Hilfe bei Social Media benötigt" ist widerrum kein guter Betreff. Wenn Sie Schwierigkeiten haben einen Betreff zu formulieren, können Sie auch Stichworte aneinanderreihen. Zum Beispiel: "Facebook, Leadgenerierung, Mittelstand, Agrar".

Frage stellen: Je mehr Informationen Sie zur Verfügung stellen, desto besser können die Social Media Berater antworten.

Komplizierte Fragen: Komplizierte Fragen erfordern lange Antworten. Wenn sie eine komplizierte Frage haben kann es schnell passieren, dass sie mehrere Fragen stellen. In dem Fall empfehlen wir das Problem in mehrere Fragen aufzuteilen. Sie werden eher eine Antwort erhalten, wenn die Frage ein klar definiertes Thema anspricht.

Wie bestimme ich den Wert der Antwort?

Der Wert der Antwort ist abhängig vom Komplexitätsgrad der Frage. Sie können auf schwierige Fragen natürlich auch 29€ bieten, allerdings kann es dann passieren, dass sich kein Social Media Berater mit der Frage beschäftigt. In diesem Fall können Sie die Frage auch neu einstellen und einen höheren Preis bieten.
